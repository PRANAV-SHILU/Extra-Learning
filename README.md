# Extra Learning Projects

This repository serves as a collection of various mini-projects and demonstrations, showcasing specific technologies, libraries, and integrations. 

## Projects Overview

| Project Name | Description | Link |
| --- | --- | --- |
| **Google-Auth** | Demonstrates how to integrate Google Authentication in a React application using `@react-oauth/google` and `jwt-decode`. | [View Project](./Google-Auth) |
| **Drag & Drop** | Demonstrates how to implement a drag-and-drop file upload feature in a React application using `react-dropzone`. | [View Project](./Drag%20&%20Drop) |
| **Razorpay** | Demonstrates the integration of the Razorpay payment gateway utilizing a Node.js backend. | [View Project](./Razorpay) |
| **fonts** | A simple demonstration of incorporating custom fonts into an HTML template. | [View Project](./fonts) |

## Getting Started

Navigate to any of the specific project directories to find their respective source code. Projects like **Google-Auth**, **Drag & Drop**, and **Razorpay** have their own dedicated `README.md` files explaining their step-by-step setup and implementation.
